# Endless Runner

A simple endless runner game built with HTML, CSS, and JavaScript.

## How to Play

- **Click anywhere on the screen** to make the character jump.
- Avoid the obstacles to keep the game going.
- The game ends when you collide with an obstacle.

## Live Demo

You can play the game live at: [https://lewiski101.github.io/endless-runner/](https://lewiski101.github.io/endless-runner/)

## Local Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/lewiski101/endless-runner.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd endless-runner
    ```
3.  Open the `index.html` file in your browser to play the game.
